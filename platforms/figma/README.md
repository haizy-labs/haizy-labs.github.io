# Mermaid Pro: Visual Editor for Figma

A powerful plugin for Figma that allows you to generate and **visually edit** Mermaid diagrams directly on your canvas!

<p align="center">
  <img src="https://haizy-labs.github.io/assets/mermaid-figma.gif" alt="Mermaid Editor Pro Figma Demo" width="800" />
</p>

## ✨ Features

- **👀 Native Figma Rendering**: Convert your Mermaid `.mmd` syntax directly into native Figma vector layers.
- **🎨 Visual Styling Integration**: Use Figma's native design tools or our intuitive UI to pick background and border colors, perfectly synchronized with your code.
- **🔄 Live Sync**: Update your Mermaid syntax and watch the Figma components update in real-time, or edit visually and get the code updated automatically.
- **🌗 Multiple Themes**: Supports all built-in Mermaid themes.

## 🚀 How to use

1. Install the plugin from the **Figma Community**.
2. Run **Mermaid Editor Pro** from your plugins menu.
3. Paste or write your Mermaid code in the editor panel.
4. Click **Render** to place the vector diagram on your canvas.
5. Select elements to visually adjust their styles, and watch the code update!

## 🛠️ Requirements

- Figma Desktop App or Figma in Browser.
