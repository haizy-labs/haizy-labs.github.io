# Mermaid Pro: Visual Editor for JetBrains IDEs

A powerful plugin for JetBrains IDEs (IntelliJ IDEA, WebStorm, PyCharm, GoLand, etc.) that not only previews Mermaid diagrams but allows you to **visually edit them** directly within your workspace!

## ✨ Features

- **👀 Native IDE Integration**: Real-time preview of your `.mmd` or Markdown files with a dedicated tool window.
- **🎨 Visual Node Editor**: Click on any node in the flowchart to visually pick background and border colors. The plugin will automatically write the correct `style` syntax back to your source code!
- **🔄 Direction Toggles**: One-click buttons to change the layout direction without touching the code.
- **🌗 Theme Switcher**: Instantly switch between Mermaid's built-in themes to match your IDE's dark or light mode.
- **💾 High-Quality Export**: 
  - **SVG**: Export perfectly cropped vector graphics.
  - **PNG/PDF**: Export high-resolution images for your documentation.

## 🚀 How to use

1. Install the plugin from the **JetBrains Marketplace**.
2. Open any `.mmd` or `.mermaid` file in your IDE.
3. The Mermaid Preview tool window will automatically open and render your diagram.
4. **To Edit Colors**: Simply click on any node in the preview! A color picker panel will appear. Pick your colors and watch the source code update.

## 🛠️ Requirements

- Compatible with JetBrains IDEs version 2023.1 or higher.
